# Relembrando comandos Git/Github

## Configurações
Adicione o nome do desenvolvedor:
```bash
git config --global user.name "Gustavo Vecchi Ruiz"
```
Adiciona o e-mail do desenvolvedor:
```bash
git config --global user.email gustavovruiz2025@gmail.com
```

## Comando Git
INIT - "Inicializar um repositório no git"
ADD - "Adicionar arquivos para a área de preparação (staging area)"
STATUS - "Verificar o estado do repositório"
COMMIT - "Salvar as alterações no histórico do projeto"
